# TypeScriptFrontend

Shopify embedded app for Ciwi Translator.

For AI agents and code-navigation guidance, read [`AGENTS.md`](./AGENTS.md)
before changing code.

## Common Commands

- `npm run dev` - start Shopify app dev on port 8080.
- `npm run build` - build the Remix/Vite app.
- `npm run worker:build` - build the translation worker.
- `npx prisma generate` - generate Prisma client.
- `npx prisma validate` - validate Prisma schema.
- `npm run check:filter` - verify translation filter sync.

Do not copy values from `.env*` into logs, docs, commits, or chat output.
